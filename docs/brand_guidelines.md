## Typography

### Primary Typeface
**Display:** Space Grotesk (Bold, 600-700 weight)
- Use for: Headlines, hero text, brand statements
- Alternatives: Inter, Outfit

### Secondary Typeface
**Body:** Inter (Regular, 400-500 weight)
- Use for: Body copy, descriptions, UI text
- Alternatives: Roboto, Open Sans

### Monospace
**Code:** JetBrains Mono

## Logo Usage

### Clear Space
Maintain clear space around the logo equal to the height of the "A" in ATLAS on all sides.

### Minimum Size
- Digital: 120px width minimum
- Print: 1 inch width minimum

### Approved Variations
1. **Full Color** - Primary logo with gradient (on dark backgrounds)
2. **Monochrome Cyan** - Single color version (#00d4ff)
3. **White** - For use on colored backgrounds
4. **Dark** - For use on light backgrounds (#0a0f1a)

### Logo Don'ts
- Don't stretch or distort
- Don't change colors outside approved palette
- Don't add effects (shadows, outlines)
- Don't place on busy backgrounds
- Don't rotate the logo

## Visual Elements

### Geometric Patterns
- Hexagons (representing interconnected systems)
- Triangular tessellations (representing AI/neural networks)
- Orbital rings (representing automation cycles)
- Circuit board patterns (subtle backgrounds)

### Gradients
Primary gradient: Linear 135deg from #00d4ff to #7c3aed to #ec4899

```css
background: linear-gradient(135deg, #00d4ff 0%, #7c3aed 50%, #ec4899 100%);
```

### Shadows & Depth
```css
/* Subtle glow effect */
box-shadow: 0 0 20px rgba(0, 212, 255, 0.3);

/* Card elevation */
box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.3),
            0 2px 4px -2px rgba(0, 0, 0, 0.2);
```

### Border Radius
- Buttons: 8px
- Cards: 12px
- Modals: 16px
- Pills/Tags: 9999px

## Voice & Tone

### Brand Voice
- **Authoritative** - We are experts in AI automation
- **Innovative** - We push boundaries and pioneer solutions
- **Accessible** - Complex technology, simple communication
- **Trustworthy** - Reliability and security are paramount

### Writing Guidelines
- Use active voice
- Keep sentences concise
- Avoid jargon when possible; explain technical terms when necessary
- Lead with benefits, follow with features
- Use "we" for the company, "you" for customers

### Key Messages
1. "ATLAS is the AI factory that builds AI factories."
2. "From prompt to production in minutes, not months."
3. "Domain expertise meets infinite automation."
4. "Your research, our intelligence, unlimited potential."
